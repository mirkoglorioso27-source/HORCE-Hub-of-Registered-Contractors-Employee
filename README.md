# HORCE-Hub-of-Registered-Contractors-Employee
HORCE - Hub of Registered Contractors &amp; Employee, è un sistema di Gestione per appaltatori e sub-appaltatori che permette di gestire tutti i documenti delle aziende come DUVRI, RAMS, DURC, CCIAA e documenti dei lavoratori come UNILAV e attestati formativi degli stessi in maniera veloce e semplice.
